# 📝 Simple To-Do App (Python Console)

A beginner-friendly **Python console-based To-Do application** where you can add, update, remove, view, and mark tasks as done/undone.  
This project is a great starting point for practicing Python fundamentals like lists, loops, conditionals, and functions. 🚀

---

## ✨ Features
- ➕ **Add Tasks** – Add new tasks to your to-do list.  
- ✏️ **Update Tasks** – Update/rename existing tasks.  
- 🗑️ **Remove Tasks** – Delete tasks you no longer need.  
- 📋 **View Tasks** – Display all tasks with their status (✅ Done / ❌ Not Done).  
- ✅ **Mark as Done/Undone** – Toggle tasks between done and not done.  
- 👋 **Exit Option** – Quit the app anytime.  

---

## 🛠️ Technologies Used
- **Python 3.x**

## ▶️ How to Run

1. Clone this repository or download the file:
   ```
   git clone https://github.com/your-username/todo-app-python.git
2. Navigate into the project folder:
   ```
   cd todo-app-python
3. Run the Python script:
  ```
  python todo.py
  ```
